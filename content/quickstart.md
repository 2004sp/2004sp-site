## Dependencies

You need two things installed before running the script:

- **Git** — [git-scm.com](https://git-scm.com/)
- **Node.js** — [nodejs.org](https://nodejs.org/)

Install both, then continue below.

## Install with the 2004sp-launcher

Check out our launcher for the easiest install in just a few minutes.

## Install with the AIO Script

Download and run the all-in-one install script from the releases page:

[github.com/2004sp/2004sp-progressive/releases/tag/install-script](https://github.com/2004sp/2004sp-progressive/releases/tag/install-script)

The script clones the repository, pulls the required server files, and wires everything together. When it finishes, your server folder is ready to run.

## Starting the Server

Once the AIO script has completed:

1. Open the server folder.
2. Run `start.bat`.
3. Log in using the standard 2004 client.
4. Bots will begin spawning and progressing within seconds.

## Getting Help

Join the Discord server for setup support. The #setup channel has pinned guides for common issues including Node version compatibility and Windows Defender false positives on the client JAR.
