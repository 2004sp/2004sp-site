# Download & Play

2004SP Progressive runs entirely on your local machine. No internet connection required after setup, no accounts, no external servers.

## Dependencies

Install these before running the setup script:

- **Git** — [git-scm.com](https://git-scm.com/)
- **Node.js** — [nodejs.org](https://nodejs.org/)

## Install

Download the all-in-one install script from the releases page — it clones the repository, pulls the server files, and sets everything up automatically:

[github.com/2004sp/2004sp-progressive/releases/tag/install-script](https://github.com/2004sp/2004sp-progressive/releases/tag/install-script)

Run it once. When it completes, the server is ready.

## Play

1. Open the server folder and run `start.bat`.
2. Log in using the standard 2004 RuneScape client.
3. Within seconds, bots will appear at Lumbridge and begin progressing.

The bots treat you as part of the world. They will trade with you, fight you in the Wilderness, and talk to you using context-aware phrases.

## Multiple Clients

The server supports multiple simultaneous connections. Run a second client to play as a second character or observe the world from a different perspective.

Your character's stats, items, and quest completions are saved locally and persist between sessions.
