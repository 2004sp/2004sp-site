# Hiscores

The hiscores table tracks XP across all 19 skills for every character on the server — both your account and the AI bots. Rankings are calculated from total XP accumulated during each server session.

## How It Works

While the server is running, XP is tracked internally for all characters. When the server shuts down, the hiscores table is written and saved. You can view the leaderboard after each session to see where you rank against the bots.

The hiscores reflect the full history of each character's progression — every skill, total level, and overall XP — making it easy to track your improvement across sessions.

## Competing Against Bots

Bots start at level 1 and progress at realistic speeds across all skills. The hiscores give you a clear picture of how your training pace compares to the bot population.

Over time, the highest-level bots will begin placing above casual players in individual skills. If you want to hold a top rank in Woodcutting or Fishing, you need to actually grind it.

## Skill-by-Skill Rankings

Rankings are available per skill and as an overall total level table. The overall table weights all skills equally — a level 99 in any skill counts the same regardless of which one.

## Viewing the Hiscores

Access the hiscores via the built-in web interface while the server is running, or check the saved output file in your server folder after shutdown. Both formats list rank, character name, level, and XP for each tracked skill.
