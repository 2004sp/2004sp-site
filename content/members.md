# RuneScape Members — Extra Content

The **Extras** repository provides additional content that extends the base Lost City server experience. Think of it as the members area — bonus content, additional scripts, and extended world events beyond what the core Progressive package includes.

## What Extras Provides

Members content in classic RuneScape opened up new skills, quests, and areas. The Extras repository follows the same philosophy — everything here is optional but enriches the world significantly.

- Additional bot behaviour scripts covering members-only skills and areas
- Extended quest support including Troll Stronghold, Death Plateau, and members quests
- Custom world events — holiday events, rare spawns, and dynamic weather behaviour
- Expanded trading economy with additional item types and price fluctuations
- Members-area navigation: bots traversing Kandarin, Morytania, and the Fremennik Provinces

## Installing Extras

Copy the contents of the Extras repository into your server folder alongside the Progressive files. The system detects the additional scripts and activates them automatically on the next server start.

No configuration is required. Extra content slots into the existing bot behaviour framework seamlessly.

## Community Contributions

The Extras repository is the most community-contributed part of the project. If you have written a bot behaviour script, a world event, or an extended quest handler — pull requests are open and reviewed regularly.

Visit [github.com/2004sp/extras](https://github.com/2004sp/extras) to browse or contribute.
